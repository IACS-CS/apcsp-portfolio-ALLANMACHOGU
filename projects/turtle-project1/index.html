Project: Turtle Mini Project

Back to Portfolio

Assignment: Create a drawing built on the turtle graphics.

For this project, we were tasked with using the turtle code to create a drawing. The requirements for this project were having parameters for the drawing, things like size, color, and structure of the drawing--where the user can integrate with the drawing parameters to see how it changes.

My drawing, which I worked on with a group, as of the character Steve in Minecraft. We had the parameter set for the size where the user could interact to see how it goes.
